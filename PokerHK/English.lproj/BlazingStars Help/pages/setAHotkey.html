Setting a hotkey.

Setting a hotkey is a simple process. Almost any key combination can be used as a hotkey, except for system-wide shortcuts (such as Cmd-Opt-Escape, the system-wide keyboard shortcut to open the "Force Quit" window).

To record a hotkey:

  1. Click on a hotkey field. Fields which have not yet been set will display "Click to record shortcut".
  2. The field text will change to "Type Shortcut".
  3. Type the key combination you want to use.
  4. The field will display the key combination you entered. This means that the hotkey is now registered.
